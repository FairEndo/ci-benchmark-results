# CI Benchmark Results

This repository stores benchmark results generated by [toolevaluatin](https://github.com/FairEndo/toolevaluatin).

- **`results/raw/`** — One JSON file per benchmark run (append-only history)
- **`results/summary.md`** — Auto-generated leaderboard (latest per provider/runner)
- **`docs/`** — Interactive dashboard (deploy via GitHub Pages)

## Dashboard

Enable GitHub Pages (Settings → Pages → Source: `main`, folder: `/docs`) to view the dashboard.
